## 在线Markdown转换器

可以用来快速的将Markdown文章转换到微信公众号支持的格式，该工具基于[online-markdown](https://github.com/barretlee/online-markdown)修改所得。

## 样例

点击链接可以直接使用：https://thinkerchan.com/markdown/

## 改进内容：

- 支持左右两栏布局，左侧用于编辑，右侧实时展现预览结果
- 更新了Webpack配置
- 优化微信公众号文章的样式(行距,文字对齐,图片标题等央视)
